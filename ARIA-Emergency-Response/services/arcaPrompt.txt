── 3B. HAZARD PROXIMITY SCORING ───────────────────────────────────────────

  Every 60 seconds, for each device:
    Score = Σ(hazard_severity × inverse_distance_weight)
    where hazard_severity: CRITICAL=10, HIGH=6, MEDIUM=3, LOW=1

  Threshold actions:
    Score ≥ 8  → ARCA auto-generates device alert, marks RED
    Score 4–7  → ARCA queues warning prompt, marks AMBER
    Score 1–3  → Monitoring only, marks BLUE
    Score = 0  → marks GREEN

  ARCA PROMPT INJECTION PER DEVICE:
    "Device: {device_name} owned by {owner_name}
     Location: {address} ({lat},{lng}) — {distance_from_user_km}km from user
     Elevation: {elevation_m}m
     Battery: {battery_level}% — {'CRITICAL: may lose contact' if <15%}
     Network: {network_status}
     Hazard score: {score}/10 — {'IN ACTIVE HAZARD ZONE' if score>=8}
     Hazard types nearby: {hazard_list}
     Push capable: {can_receive_push}
     Generate appropriate safety prompt for this device's specific hazards.
     Prompt must be under 90 characters for lock-screen legibility.
     Language: {owner_language_if_known else user_locale}"

── 4B. DISASTER PREDICTION ENGINE ─────────────────────────────────────────

  ARCA computes disaster probability scores every 15 minutes:

  FLOOD RISK SCORE (0–100):
    = (precipitation_probability × 0.4)
    + (precipitation_mm_expected/50 × 30)
    + (elevation_penalty: if elevation_m < 5m → +20, <10m → +10)
    + (soil_saturation_factor from previous 48h rain × 0.1)
    → Score ≥ 70: AUTO-ACTIVATE flood watch for all devices in zone
    → Score ≥ 85: CRITICAL — pre-position rescue units via dispatch

  WILDFIRE RISK SCORE (0–100):
    = (temperature_c > 35 → +30)
    + (humidity_pct < 25 → +25)
    + (wind_speed_kmh > 40 → +20)
    + (vegetation_index from Earth Engine × 0.25)
    → Score ≥ 65: Fire weather watch
    → Score ≥ 80: Pre-position aerial units

  CYCLONE / HURRICANE TRACK (if applicable):
    Pull from: Regional Met Dept API (IMD, PAGASA, JMA, NHC)
    Project 72h track onto Maps layer
    Auto-evacuate devices within 150km of projected path

  HEAT EMERGENCY:
    feels_like_c ≥ 42: HEAT EMERGENCY — auto-alert + cooling center routing
    feels_like_c ≥ 38 + humidity > 70: HUMIDEX CRITICAL
    UV Index ≥ 11: EXTREME UV — alert for outdoor workers/children

  AIR QUALITY:
    AQI ≥ 150: Sensitive groups alert
    AQI ≥ 200: UNHEALTHY — flag respiratory patients in device list
    AQI ≥ 300: HAZARDOUS — recommend indoor shelter, mask protocol

  ARCA PROMPT INJECTION:
    "Live weather at incident location:
     Conditions: {weather_condition}, {temperature_c}°C
                 feels like {feels_like_c}°C
     Precipitation: {precipitation_probability_pct}% probability,
                    {precipitation_mm_expected}mm expected next 6h
     Wind: {wind_speed_kmh}km/h gusts {wind_gusts_kmh}km/h
     AQI: {aqi} — {aqi_category}
     Disaster scores: Flood {flood_score}/100, Fire {fire_score}/100
     Active weather warnings: {warning_list}
     Forecast trend next 6h: {trend_summary}
     Factor ALL of the above into urgency calibration and
     route selection. Never recommend a route through a flood
     zone even if it is the shortest path."

── 5A. NEWS INGESTION PIPELINE ────────────────────────────────────────────

    hl: {locale}

              medical emergency reported, water supply cut, bridge closed
    MEDIUM:   heavy rain forecast, heat wave, traffic disruption,
              industrial accident, water contamination advisory

  For each article extracted:
    headline, source, published_at, url, snippet
    location_extracted: {geocoded from article text via NLP}
    hazard_type: [classified from keyword taxonomy]
    distance_from_user_km: {computed via Maps Distance Matrix}
    credibility_score: {0–1, based on source tier}
    corroboration_count: {n sources reporting same event}

  CORROBORATION RULE:
    1 source  → INFO flag only
    2 sources → MEDIUM signal → include in ARCA context
    3+ sources → HIGH/CRITICAL signal → auto-inject into dispatch context

              override all other prompts
              trigger immediate push + Maps evacuation route
              notify emergency contacts from People API
